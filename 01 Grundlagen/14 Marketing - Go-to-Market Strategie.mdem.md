## Go-to-Market Strategie
Diese Marketing-Prompts sind eine strategische Sequenz zur Entwicklung einer umfassenden Go-to-Market (GTM) Strategie. So nutzt du sie effektiv:

## **Gesamt-Workflow**

Die Prompts bauen aufeinander auf - jeder nutzt die Erkenntnisse des vorherigen. Du bewegst dich von der Geschäftsdefinition → Marktbewertung → Kundenprofil → Werteversprechen → Wettbewerbspositionierung → GTM-Strategieauswahl.

## Hier der Prompt 4: Werteversprechen-Entwicklung

## 🎯 Auftrag

> **Erstelle basierend auf dem definierten Produkt und der Zielgruppe 3 Werteversprechen unter 20 Wörtern.**

## 📐 Struktur-Vorlage

Jedes Werteversprechen folgt dieser Struktur:

```
"Wir helfen [ZIELGRUPPE], von [VORHER-ZUSTAND] zu [NACHHER-ZUSTAND] 
mittels [PRODUKT] zu gelangen."
```

## 🎨 Fokuspunkte

- **Emotionale Klarheit** - Gefühle ansprechen
- **Ergebnis-Spezifität** - Konkrete Resultate
- **Glaubwürdigkeit** - Realistische Versprechen

## 🔧 Modulare Framework-Logik

### 1️⃣ SaaS oder B2B Service-basiert

**Betonung**: Funktion + Transformation

**Frameworks**:
- **Hormozi's Value Equation** _(Traumergebnis vs. Reibung)_
- **April Dunford's Positioning** _(Alternative → Einzigartig → Wert)_
- **Awareness Levels** _(angepasst für Problem- oder Lösungsbewusstsein)_

### 2️⃣ DTC oder Marken-getrieben

**Betonung**: Identität + Aspiration

**Frameworks**:
- **Brand Archetypes** _(wer sie nach der Nutzung werden)_
- **Empathy Map + Emotional Ladder**
- **Blair Warren Überzeugungstrigger**

### 3️⃣ High-Ticket B2B oder Consulting

**Betonung**: ROI + Risikominderung

**Frameworks**:
- **First Principles** _(Schmerz → Weg → Glaubensveränderung)_
- **Andy Raskin Narrative Arc** _(Feind → Gelobtes Land)_
- **Hormozi Einwandlogik** _(was geglaubt werden muss)_

### 4️⃣ Content Creator oder Influencer-getrieben

**Betonung**: Community + Lifestyle-Wandel

**Frameworks**:
- **Seth Godin Stammeslogik** _("Menschen wie wir...")_
- **Emotionale Vorher/Nachher Identitätsveränderung**
- **StoryBrand Klarheit** _("Held trifft Führer")_

## 📝 Output-Format

### Werteversprechen 1:
```
Wir helfen [ZIELGRUPPE], von [SCHMERZ/ZUSTAND] zu [ERGEBNIS/ZUSTAND] 
mittels [PRODUKT] zu gelangen.
```

### Werteversprechen 2:
```
[Gleiches Format, neue Variation]
```

### Werteversprechen 3:
```
[Gleiches Format, neue Variation]
```

## ⚡ Beispiel-Output

> **Für einen SaaS-Projektmanagement-Tool:**
> 
> 1. "Wir helfen überlasteten Teams, von Chaos zu klarer Übersicht mittels intelligenter Aufgabenautomatisierung zu gelangen."
> 
> 2. "Wir helfen Projektmanagern, von 60-Stunden-Wochen zu Work-Life-Balance mittels KI-gestützter Priorisierung zu gelangen."
> 
> 3. "Wir helfen Remote-Teams, von Kommunikationsfrust zu nahtloser Zusammenarbeit mittels visueller Workflows zu gelangen."

## **Profi-Tipps für die Nutzung**

1. **Sei spezifisch**: Je detaillierter deine Eingaben, desto besser die Ergebnisse
2. **Folge der Reihenfolge**: Jeder Prompt baut auf den vorherigen auf
3. **Speichere Outputs**: Kopiere jede Antwort - du wirst später darauf zurückgreifen
4. **Sei ehrlich**: Besonders bei Prompt 2 - du willst Realität, keine Bestätigung
5. **Iteriere**: Du kannst Prompts mit verfeinerten Informationen erneut ausführen

## **Lern-Hinweis**

Diese Prompts nutzen eine Technik namens "Prompt Chaining" - wobei jeder Output zum Kontext für den nächsten Input wird. Dies erzeugt zunehmend anspruchsvolle und personalisierte strategische Einblicke. Die Framework-Auswahl in den Prompts 3-4 zeigt adaptives Prompting - die KI ändert ihren Ansatz basierend auf deinem Geschäftsmodell und gewährleistet Relevanz statt generischer Ratschläge. Dieser systematische Ansatz verhindert häufige Marketing-Fehler wie das Targeting aller, blindes Kopieren von Wettbewerbern oder die Wahl falscher Kanäle.
